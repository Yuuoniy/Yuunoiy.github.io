windows10 下升级 node

Posted on 2017-09-02 |

1.去官网找到对应的msi文件

2.下载，安装到node历史安装目录
3.查看是否升级成功，完毕

尝试过通过 n 来安装node最新版，但是 n 一直不能安装成功，也尝试过用 cnpm 安装 n ，出现了自己无法解决的问题。最终还是去官网下载 msi 文件
另外遇到安装是窗口在屏幕外一直无法拖动的情况，右键选择移动（M键）后使用键盘的上下左右移动后回车就好了 ><
